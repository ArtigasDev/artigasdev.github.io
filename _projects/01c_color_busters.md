# Features
- Full Game cicle (main menu, game loop, game over...).
- Layer Collision Matrix configuration.
- Score System.
- URP configuration.
- Cinemachine asset.
- Audio Manager.
- Unity's Particle System.
- SENSATION (UI movement according to Score multiplier, Screen shake, ...).
