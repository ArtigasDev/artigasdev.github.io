# Features
- Transparecy materials.
- Frustrum Culling.
- Cubemap.
- Enviroment Fog.
- Phong Lighting model using forward rendering.
- Light creation system (Directional Light, Point Light, Spotlight).
- Procedurally generated city.
- Flying camera.
