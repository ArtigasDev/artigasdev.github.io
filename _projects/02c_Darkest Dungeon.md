# Features
- Full Game cicle (main menu, game loop, game over...).
- Scriptable Objects. (Character creation, Inventory items).
- Turn-based Combat System.
- Inventory System.
- Enemies Pooling.
- Interactive UI (inventory items, torchlight, map, ...).
